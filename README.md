# Aveji Figma 
Live Demo - [Ознакомиться](https://bobicentr.github.io/avejifigma/)

Небольшой учебный проект для практики вёрстки на HTML и CSS.
# Главные фичи
- Полный адаптив, сайт удобно просматривается как на мобильных устройствах, так и на пк.
- Необычное расположение элементов, используя CSS Grid
- Небольшие креативные анимации CSS при наведении на элементы
