# О проекте
Aveji -- лендинговый сайт-визитка для компании по производству мебели. Адаптивная вёрстка на чистых HTML и CSS с использованием Flexbox и Grid.
# Технологии 
- HTML
- CSS
# Основные особенности
- Полный адаптив, сайт удобно просматривается как на мобильных устройствах, так и на пк.
- Необычное расположение элементов, используя CSS Grid
- Небольшие креативные анимации на CSS
